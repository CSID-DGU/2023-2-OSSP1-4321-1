## 실행 방법 
실행방법
1. 프로젝트를 로컬 레포지토리로 clone합니다.
2. intellij IDE, java 17을 권장합니다. 
3. `src` > `main` > `resources` 디렉토리에 application.properties 파일을 작성해 추가하세요
4. mysql 세팅 완료 및 실행
5. 파이썬 모듈 추가
6. 프로젝트 실행
